Seattle Public Life: Mapping Projects

This project uses data from the Seattle Public Life Study to map changes in public life from 2018 to 2023. We look at the distribution of race, gender, and age groups in different neightborhoods of Seattle. This page contains all the interactive plots. You can see more detailed information about each observation site by moving your cursor to the corresponding dots.
